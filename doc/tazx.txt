Les boîtes de dialogues sont propulsées par dialog, il vous
suffit d'utiliser les flèches pour séléctionner une valeure, et
ensuite de valider votre choix en pressant sur <ENTER>. Vous
pouvez démarrer une session graphique en séléctionnant le mode
video, qui convient le mieux à votre matériel.

Après la première exécution de startx, la configuration est
sauvegardée dans le fichier exécutable ~/.xsession, que vous
pouvez modifier avec un éditeur de texte. A noter que vous
pouvez aussi lancer un session manuellement avec la commande :

    $ Xvesa -ac -shadow -screen 1024x768x16 & jwm

Pour plus d'informations, vous trouverez la documentation de
SliTaz dans: /usr/share/doc/slitaz. Un page du Handbook est
dédiée à la configuration du système X. Sur le site Web de
SliTaz : http://www.slitaz.org/doc/handbook/
